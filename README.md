# about-me

ポートフォリオサイトです。Next.jsで作られています。

![ホーム画面](./docs/images/home.png)

## 特徴

- カスタムカーソル
- シェーダーエフェクト
- グラインオーバーレイ
- マグネティックボタン
- スムーズなスクロールナビゲーション

## セクション

- **Hero**: トップセクション
- **About**: 自己紹介とスキル
- **Work**: 制作物一覧
- **Contact**: お問い合わせ

## 開発

```bash
# 開発サーバー起動
pnpm dev

# ビルド
pnpm build

# 本番起動
pnpm start

# リント
pnpm lint

# フォーマット
pnpm format
```

## 技術スタック

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- Biome (Linter/Formatter)
